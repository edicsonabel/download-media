Download anything

A friendly yt-dlp wrapper with a step-by-step wizard, smart clipboard detection, carousel item picker, and zero learning curve.

Install now View on GitHub
$ yay -S download-media

Built for humans, not flags

Every annoying yt-dlp command you keep googling — turned into a few keystrokes.

Step-by-step wizard

URL → type → quality → format → folder → cookies. Type b to go back, c to cancel.

Smart clipboard

Run with no args. If you copied a URL, it offers to use it. No more pasting.

Carousel picker

Instagram carousel? Pick exactly which slides you want: 1-3, 1,2,4, or all.

Preview before download

Title, channel, duration, size — see what you're about to grab before it starts.

Browser cookies

--cookies brave works for YouTube logged-in, Instagram private, age-restricted videos.

1800+ sites

YouTube, Twitter/X, Instagram, TikTok, Facebook, Twitch, Vimeo, Reddit — anything yt-dlp supports.

Localized output

Respects your XDG dirs — saves to Vídeos, Música, Movies, or wherever you set them.

Equivalent command hint

After every wizard run, prints the exact CLI command — so you can repeat without prompts.

Get started in 3 steps

From zero to first download in under a minute.

1

Install from AUR

Use your favorite AUR helper:

yay -S download-media
2

Run the wizard

No flags needed. It asks everything:

download-media
3

Or jump straight in

Pass a URL to skip the wizard:

download-media -a "https://..."

Recipes

Common things you might want to do.

YouTube video at 1080p 
# mp4 → ~/Vídeos
download-media "https://youtu.be/dQw4w9WgXcQ"
Just the audio (mp3) 
download-media -a "https://youtu.be/dQw4w9WgXcQ"
Instagram with browser cookies 
download-media --cookies brave \
  "https://instagram.com/p/xxx/"
Pick items from a carousel 
# Wizard shows the list, you type "1,3,5"
download-media "https://instagram.com/p/xxx/"
With embedded subtitles 
download-media -s --sub-langs en,es \
  "https://youtu.be/xxx"
Multiple URLs in one go 
download-media "url1" "url2" "url3"
